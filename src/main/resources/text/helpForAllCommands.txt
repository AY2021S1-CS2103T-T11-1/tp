
Contact commands
----------------------
contact add	                Adds a contact                                              contact add n/NAME p/PHONE_NUMBER e/EMAIL a/ADDRESS [t/TAG]… [r/REMARK]
contact edit                Edits an existing contact                                   contact edit INDEX [n/NAME] [p/PHONE] [e/EMAIL] [a/ADDRESS] [t/TAG]… [r/REMARK]
contact list                Lists all contacts                                          contact list
contact find                Find contacts given search keyword(s)                       contact find KEYWORD [MORE_KEYWORDS]
contact delete              Deletes a contact                                           contact delete INDEX
contact sort                Sorts contact list                                          contact sort KEYWORD [ORDER]

Archive commands
----------------------
archive add                 Sends a contact to the archive                              archive add INDEX
archive list                Lists all archived contacts                                 archive list
archive remove              Removes a contact from the archive                          archive remove INDEX

Tag commands
----------------------
tag list                    Lists all tags                                              tag list
tag edit                    Edits an existing tag to the specified tag name.            tag edit INDEX n/NAME
tag delete                  Deletes the specified tag                                   tag delete INDEX
tag find                    Displays all entries related to the specified tag           tag find INDEX [MODEL]

Sale commands
----------------------
sale list                   Lists all sale items                                        sale list [c/CONTACT_INDEX]
sale add                    Adds a sale to a customer                                   sale add c/CONTACT_INDEX… n/ITEM_NAME d/DATETIME_OF_PURCHASE p/UNIT_PRICE q/QUANTITY
sale delete	                Deletes a sale item of a customer                           sale delete s/SALE_INDEX…
sale edit                   Edits an existing sale.                                     sale edit c/CONTACT_INDEX… n/ITEM_NAME d/DATETIME_OF_PURCHASE p/UNIT_PRICE q/QUANTITY
sale breakdown              Displays a breakdown of sales by sale tags                  sale breakdown

Meeting commands
----------------------
meeting add	                Adds a meeting with the specified contact                   meeting add CONTACT_INDEX m/MESSAGE d/START_DATETIME du/DURATION
meeting edit                Edits an existing meeting                                   meeting edit INDEX [c/CONTACT_INDEX] [m/MESSAGE] [d/DATETIME] [du/DURATION]
meeting list                Lists all meetings.                                         meeting list [CONTACT_INDEX] [a/]
meeting delete              Deletes the specified meeting                               meeting delete INDEX

Reminder commands
----------------------
reminder add                Adds a reminder with the specified contact                  reminder add CONTACT_INDEX m/MESSAGE d/DATETIME
reminder edit               Edits an existing reminder                                  reminder edit INDEX [c/CONTACT_INDEX] [m/MESSAGE] [d/DATETIME] [st/STATUS]
reminder list               Lists all reminders                                         reminder list
reminder delete	            Deletes the specified reminder                              reminder delete INDEX

Other commands
----------------------
clear                       Clears chat bot history                                     clear
purge                       Clears all data                                             purge
exit                        exits program                                               exit
